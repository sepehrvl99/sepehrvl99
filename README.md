# Telegram Keyword Alert Bot (Aiogram v3)

## پیش‌نیازها
- Python 3.8+
- Telegram Bot Token (از BotFather)

## نصب
1. ساخت و فعال‌سازی محیط مجازی:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```
2. نصب وابستگی‌ها:
   ```bash
   pip install -r requirements.txt
   ```

## پیکربندی
- متغیر محیطی `BOT_TOKEN` را با مقدار توکن ربات خود تنظیم کنید؛ برای مثال در لینوکس:
  ```bash
  export BOT_TOKEN="123456:ABC"
  ```
- فایل `data.json` در اولین اجرا به صورت خودکار ساخته می‌شود و اطلاعات کاربران، گروه‌ها و کلیدواژه‌ها را نگه می‌دارد. این فایل در `.gitignore` قرار گرفته است تا داده‌های حساس وارد مخزن نشوند.

## استفاده
- اجرای بات:
  ```bash
  python bot.py
  ```

### دستورات
- `/start` یا `/register`: ثبت‌نام برای دریافت هشدارها
- `/addgroup <group_id>`: افزودن گروه برای مانیتور کردن
- `/addkeyword <keyword>`: افزودن کلیدواژه برای پیگیری
- `/list`: نمایش کاربران، گروه‌ها و کلیدواژه‌های فعال

## نحوه عملکرد
بات پیام‌های متنی گروه‌های مشخص‌شده را بررسی می‌کند. در صورت یافتن هر یک از کلیدواژه‌ها، هشدار تلگرامی برای تمام کاربران ثبت‌شده ارسال می‌شود.
